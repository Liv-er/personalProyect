Recomendaciones de uso:
Hay dos formas de buscar por nombres mas o menos especficos (Huevo blanco, Azucar, Nutri leche) o por el tipo de producto que es por ejemplo: LECHE, HUEVO, SEMILLAS, REFRESCO, AGUA.
Otras recomendaciones son que solo lo busques en SINGULAR y no en plural (LECHE no leches) al mismo tiempo que la unica letra en mayusculas va a ser la primera tambien con las medidas siguientes las puedes escribir asi:
GRAMOS => gr
Piezas => pz --- piezas
MILILITROS => No los pongas tipo: Coca cola 600
LITROS => L --- l
Algunas de las categorias que hay son:
Huevo // Farolito // Semilla // Nescafe // Leche // Cerveza // Refresco // Maruchan // Agua
